## 📌 Project Description

This dashboard is based on a survey of data professionals, analyzing their experiences, challenges, and career insights in the data field.

The analysis focuses on key aspects such as:
- Difficulty of breaking into data-related roles  
- Salary distribution across different job titles  
- Popular programming languages among data professionals  
- Geographic distribution of respondents  
- Satisfaction with salary and work-life balance  

The goal of this project is to transform raw survey data into meaningful insights using interactive visualizations, helping better understand trends in the data industry.
